## Instalación
1. Clonar el repositorio:
   git clone https://github.com/DaveeTov/appgame.git
   cd appgame
2. Instalar dependencias:
   npm install
3. Ejecutar la aplicación:
   ng serve
   La app se levanta en http://localhost:4200
